# CipherLock Ultra 🔐

CipherLock Ultra is a browser-based password generator that focuses on creating strong, secure passwords while helping users understand *why* a password is secure.

Unlike a basic random password generator, this project also analyzes password strength, estimates entropy, checks passwords against a list of commonly leaked passwords, and supports different security policies used by major organizations.

The project was built using only HTML, CSS and JavaScript without any external frameworks.

---

## Features

- Generate cryptographically secure passwords
- Adjustable password length (8–128 characters)
- Uppercase, lowercase, numbers and symbols
- Optional extended symbols
- Exclude ambiguous characters (O, 0, l, I, etc.)
- Password entropy calculation
- Password strength indicator
- Estimated password crack time
- Password policy modes
  - Google
  - Microsoft
  - Banking
  - Custom
- Password compliance checker
- Dictionary check against common leaked passwords
- Password history
- Copy password to clipboard
- Export history as:
  - TXT
  - CSV
  - JSON
- Generate QR code for transferring passwords between trusted devices
- Responsive interface

## Why I Built This

Most online password generators simply create random passwords without explaining whether they are actually secure.

The goal of this project was to build a tool that not only generates passwords but also helps users understand password security through entropy calculations, policy validation and strength analysis.

It also gave me hands-on experience working with JavaScript, DOM manipulation, UI design and security-related concepts.
